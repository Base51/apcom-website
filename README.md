# APCOM

Website da **Associação Portuguesa de Combate Medieval**.

## Estrutura

```
index.html          → Homepage
clubes.html         → Clubes filiados
eventos.html        → Agenda de eventos
desporto.html       → Sobre o desporto
associacao.html     → Sobre a APCOM
contactos.html      → Contactos
css/styles.css      → Estilos custom
js/main.js          → Theme toggle + mobile menu
assets/images/      → Logos e imagens
```

## Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- GitHub Pages (deploy direto)

## Deploy

Push para `main` → GitHub Pages serve automaticamente da raiz.

## Desenvolvimento

Abre `index.html` no browser. Sem build, sem dependências.
